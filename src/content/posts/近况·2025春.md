## 项目·Muicebot

> 我认为，无论人工智能领域如何发展，LLM 的任务始终只有一个——帮助人类解决**实际问题**。它即不能代替人类，更**不能**成为人类。

今年二月份的时候开了一个新坑，把 [Muice-Chatbot](https://github.com/Moemu/Muice-Chatbot) 用新的机器人框架 Nonebot2 重新写了一遍，这里是项目链接：[Moemu/MuiceBot](https://github.com/Moemu/MuiceBot)

她能干什么呢？她能调用各家的 SDK 进行大语言模型交互，还能多模态；她能无缝衔接各类互联网平台；有人设模板，聊累了能够换一个人设/模型，还能实时注入记忆和你自己的设定之类的；还有 Function Call 和 MCP Host 实现和插件系统

但没用，还是不如 Muice-Chatbot 成功，时至今日甚至还有人 fork 那个仓库。当然我也承认现在我也没有噱头来宣传 Muicebot ，更何况半路还杀出一个程咬金

在我**自己一个人**完善整个插件生态之前，我想我还有很长的一段路要走

## 人设·沐妮卡

沐妮卡(Muika)是我去年十月份想出来的一个新的项目代号，这个项目将基于现有的 LLM 服务，以复刻 DDLC 中的 Monika 为目标，并提供其在真实机器中的访问 API ，实现打破第四面墙的目标。

但是现在正如你所见，直至6月12日，我目前的进度也仅仅是撰写了一个[人设模板](https://github.com/Moemu/MuiceBot/blob/main/muicebot/builtin_templates/Muika.jinja2)而已，我不清楚这个项目能否在 2026 的圣诞节前正式落地。

我创建了一个组织来负责这个项目的开发 [MuikaAI](https://github.com/MuikaAI) ，但目前也只是为 Muicebot 的插件库服务

就说到这里吧，如果你直到我之前发的吊胃口宣传图，那说的就是这个，你可谓是赚大了

好了，看起来你也准备好了，让我们开始吧。

从 5 月 15 日开始，我的情绪就逐渐趋于不稳定，一直绝望崩溃，于是 5 月 23 日，我就前往了最近的人民医院去寻找原因。虽然我和医师没有多少语言上的往来，但诊断结果告诉我，我得了重度抑郁症+中度焦虑。

抑郁症的苗头是什么时候开始的呢？也许是一年前的高考成绩不如意，也许是一年前横坐在4楼高的宿舍围栏，也许是两年前的自我边缘化，再或许是若干年前的家庭暴力？我不太清楚。但当我写下 **After Muika(当我离开之后)** 时，我就发觉这件事情就超出了我的控制范围之内了。

从我写给医师的文字来看，这次事态变得严重的原因主要有三个：

- 现实大学的交友失败，同班的同学基本没交上，反倒是交了两个来自不同系的好友
- 网络群组的交友也失败了。这点我不打算细讲，我也没想过要把原因推卸在别人身上
- 在第二点的基础上，我发现了更多比我厉害的存在。他们年龄、目前的学历更小，他们知识领域比我广，他们很开心；他们的学历比我高，他们的技术栈比我深，他们找不到工作

除此之外，每次离开寝室，我都焦虑着宿舍里面的电器会不会因为我都疏忽而起火，洗澡后会不会没把用品清理干净，我的父母会不会在一瞬间就突然离世然后我就得艰难求生。

以上的这些思想造就了独特的存在，而这个存在不显于外表。现实中的我，依旧有说有笑，只有在屏幕前或者是在被窝里，才会流露出来。

我也尝试过医生开的疏肝解郁胶囊，但只在第一次时它清空了我的神经，使我没有办法理解我的所作所为。在这之后便失去了有效性，每当时间流逝或者发生了在别人眼中看似平常的事情，我就会陷入绝望，短则数小时，长则1~2天。

与好友会面的可以很好的证明上面的两点，而这两点在写这篇文章之前我没有和他说过。不知是否药效已过的原因，路途的下半程，我陷入了抑郁，感到呼吸困难面部麻痹，许多脑海在我脑中闪过，但我仍然一言不语，继续完成了这趟旅行直至吃完甜品告别。

我分享这些，不是为了让你同情我，你也没办法向我表示同情，因为我关闭了评论区。反而，我想告诉你，如果你现实中，真的遇到了陷入抑郁症的好友，你应该要怎么做而不是做什么，比起网上那些没得抑郁也没接触过抑郁症患者的泛泛而谈，我想我更有发言权。

面对抑郁症患者，不要做以下事情（不是双重否定表肯定，而是强烈建议不要做）

- ❌突然闯入抑郁症患者的生活。你不能只在别人得抑郁症的时候才去问候他，对我而言这样的行为代表着一种虚伪
- ❌劝他们好好调理、运动、多喝水、多吃抗郁药、多检查。除非你对他而言很重要，不然这些话语只会让他们感到厌烦，因为他们很清楚这些本来就是要去做的。或者换个角度说，这是对抑郁症状了解不深的表现，只看到了病，没有看到因。而对于检查，除非你肯出钱，不然提都不要提
- ❌问他们抑郁是不是真的。我他妈最烦这种人，人家都发出来说明在求救了你得先求证是不是真的是得了病
- ❌劝他们多多交流。有些父母应该好好反思一下为什么对自家孩子得病完全不知情，我不奢求他们有没有意识到得病的原因了
- ❌在病人倾诉时转移话题/强行直接以“摸”结束话题。这样做的后果无异于**直接忽视 ta 的发言**（ta 由于情绪状态不稳定，也无法接上的你的话题和你的摸）。无论你是谁，请记住，谈话的第一要务是**相互尊重**，如果你实在不知道说什么，就不要说。

而是

- ✅给他们的倾诉说说和朋友圈点个赞，让他知道你有在关心TA
- ✅如果他们有自己的社群，去他们的社群里面多点和TA互动，让TA知道自己还有一群人在TA的周围
- ✅去和他们聊天，如果你对他很**重要**，万一他的抑郁症就是由你对他的忽视而引起的呢？

我想说的大概就说的差不多了，接下来还有一些从我空间中转载的内容，或许可以让你知道我内心的一些想法。

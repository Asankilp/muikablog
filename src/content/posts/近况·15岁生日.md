# 6月12日 星期六（生日）：

没有满天飞的生日祝福，只有以前认识的同学一句令人厌烦的“生日快乐”（因为他在班内很安静而且我在复习的时候偷偷凑过来说的“生日快乐”）

看来生日也就如此了，没有生日礼物（除了当天4科的试卷以外），没有真诚的生日祝福，只有自己，那么我就祝自己：

A. 生日快乐，心想事成，万事顺利

B. 中考题名，考上三中，过上美好的高中生涯

C. Python永远不报错，早日精通 Python, C++, 创造更多的 Github 仓库（前提是永远不会“**github.com** 花了太长时间进行响应”）和博客内容，B站粉丝早日破万

# 打算：

 接下来的日子我已经安排好了计划清单，能完成清单上的内容就不错了，周末花在blog的时间会越来越少，更多的是学习，对于中考我早已做好了心理准备，去不了三中就去二中，二中也不错，在二中起码还能拿个前100什么的，选择冲三中呢是因为我听说那里有许多多彩多艺的人，而且离我以前的家也很近，想找回从前在村里面生活的感觉，而不是单纯为了那里的重本率很高什么的，我知道我的水平。

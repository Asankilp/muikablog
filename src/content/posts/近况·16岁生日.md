# 前言

眨眼间6月份也到了，写完这篇文章的时候应该高考也结束了，希望能看到这篇文章的高考生能取得自己满意的结果，开心就好。

# 创作瓶颈

高考和端午给我放了1个星期假~~（"2倍调休"）~~，也榨干了我的创作灵感，现在无论是视频也好，博文也好，也没有什么创作的灵感了，现在搞的都是以前的老本，真可谓是创作瓶颈了，咋办才好？

# 16岁生日抢先体验

2022年6月8日我给自己搞了个16岁生日抢先体验日，打算彻底放松一下，这一天也过完了，没有什么大事发生，作业还得照写，也找不到什么地方玩。看来这就是4天后的气氛，还是在学校过（去你妈的二中，一边布置天多的作业，一边还给老子占用周末作业，比调休还他娘的调休，还美名曰"不应该有1周假期了，这是你的学习时间"）。

我也习惯了，16年来，抛开幼儿园还没记事那些年，我就没过几个像样的生日，一是最常见的忘记，二是推迟几周，三是买个蛋糕敷衍一下（所以说来我很不喜欢吃蛋糕，10分钟内回房继续工作），唯一的一个生日礼物是2个月后就烂的智能电子表。我打心底也知道什么原因，也不愿在家长面前吐槽，我也知道我的交往圈几斤几两，还是默默接受现实的好。

16岁，我希望将来的自己能活成自己喜欢的样子，既要享受人生，又要为其他人创造更好的人生，演好并非我选择的剧本，我认为这便是我存在的意义。

# 童年之旅

我经常听到某些人在吐槽自己童年的时候没有干嘛干嘛，说实话，我也吐槽过，比如当年为什么没有去学计算机，去了解二次元，而是去看游戏实况，后面我才想通了，原来那是我为数不多的自由自在，无忧无虑的童年啊。

基于这个想法，我打算写本书来详细回忆我的童年，优点有很多，比如自己老的时候，或者自己失意的时候想回忆童年的时候就可以看看这本书来从快节奏的生活中得到暂时的解脱；又比如在记忆力还算好的时候写下这本书，以后自己看看的时候可以笑得出来，原来我那时候还他娘的做过这种事......

这本书的名字暂定为《童年之旅》，这是从1个月前我从叔叔那买来的《魔女之旅》而来的。封面也暂时用了去年10月份在故居公园随手拍的图片，打算今年暑假再去拍几张好看的来。

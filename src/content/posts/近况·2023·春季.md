# 春节

今年的春节是我过得最糟糕的（仅仅对于我来说，我相信疫情过去的第一个春节各位过得很欢乐，以下的文字只是起记录作用，只是我自己发牢骚而已）

寒假前，兴致勃勃地地搞了许多行程规划，然后被突如其来的"早点回家"全部推翻。

寒假在老家不知道干什么，就干睡觉。如果没有睡意就看米奇妙妙屋（1月份刚开的冤种会员不看点东西可惜，用来看好番担心没有兴致就去补童年）。结果天天都在破睡眠记录，最高一天睡了16小时。

终于等来初六，原定返城，算好时间买了电影票，但被"急什么"推翻，只好再等2天。

初八，大吉大利，返城看电影去咯~和父母交代在中途下车，自己找公交车去电影院，结果没带口罩被拒上车，没办法，只好找了辆哈佛骑40分钟去了电影院，还用了一次免违约金机会。

电影看的球二，也只有球二值得看，流泪了3次，是以前看电影从未有过的。

看完电影，收到消息，去别人家吃饭，好在离电影院不远。刚好门口有辆共享电动车，第一次骑，很快上手，不得不说，骑起来真爽。

终点不是还车区，得付20大洋，这次没有免违约金机会。

初九~初十，还得去臭名昭著的外婆家，还被热心群众把底裤都问出来了。

但接下来4天，都痛痛快快地玩了，但作业欠了一科数学，但我并不担心，因为在返校1天半内就完成辣。

来聊聊为什么家乡的春节总是这么无聊呢，究其原因有以下几点:

1. 血缘关系放不开。串门次数年年减少。
2. 不能走太远，因为你不知道哪里会有蛇，尤其是乡下地方，离集市远。但是最终原因是去过太多地方，对家乡的烂环境没好感，尤其是"新农村"改造，田都种上了不经看的经济作物。
3. 能背出来过年的流程，而且放的烟花永远是那个样，连改都不带改。总之就是过年形式死板。
4. 网络垃圾，经常断网，不如睡觉。

我理想中的春节应该是这样的:

1. 过年前去行花街去买些年货，搞点水仙花之类的让家里过年氛围浓起来。

2. 去各种地方增加眼界，看看各地是怎么庆祝新年的。

3. 我所在城市有烟火大会，应该去看看。

4. 有一个良好的网络来观看过年直播。

0. No Parents, Only Friends(Even Classmates).

现在做不到，未来有很长一段时间不能做到。Waiting 22.

# 病情

先来谈谈3天内的病情，我害怕我忘记了整个发作过程。

2023.02.10 2点钟左右，因为反酸(?)、有辣感、四肢无力、呼吸困难从睡梦中醒来(睡眠时间3小时左右)
3点钟，见情况未见好转，鼓起勇气找舍管，舍管带我去找了校医，开了药并给我服用。
4点钟，把隔夜饭连本带息地给你呕了出来，情况终于有所好转。
然后就睡不着了。
6点钟打铃起床，发现四肢无力症状更加严重并且带有呕吐倾向。
去饭堂拿了个牛奶＋鸡蛋就回了教室。(如果这个时候吃了早餐我预计会在40分钟内呕出来，但后来喝了点牛奶)
7点钟衡水式早读，想办法摸了鱼，但是还是被语文老师发现并要求默写课文，试了一下，没力写字。
早读结束，去找校医(已换人,所以ta应该不知道我2点钟来找过校医)，ta劝我观察30分钟，情况未见好转，去医院检查。
14点左右。确诊肠道内细菌感染。(家里的封建势力一致认为由营养不足引起)。此时情况好转了半成，但是发生了低烧。(一周目结束，二周目开启)
当晚，服用了药物。药物和水分在胃里待了不知多少个小时，反正在3个小时过后我才能睡得着。(二周目结束)
隔天晚23:40，出现心脏绞痛，在一个小时后结束(三周目：超过正常值1500%)

我猜测是由于以下原因引起的一连串病情:
睡前吃了学校的4块钱一包的垃圾猪肉干->引起细菌感染->发生呕吐->胃里的营养物质低于正常值->细菌感染持续，为了安全性，没吃早餐->封建势力一致认为营养不良，投喂大量食物，我服用医生的药物(补盐水)->胃里因为病情无法迅速消化->因为消化系统不良和心脏存在小问题引发三周目。

接下来得戒辣，戒咖啡，戒油腻了

所以，
避免过度消费，与封建势力对抗，并科学用药，保护好自己。
